# Sistema de Controle de Estacionamento

Projeto desenvolvido em Java utilizando Swing, JDBC e MySQL, com padrão MVC e DAO.

## Funcionalidades
- Login de usuários
- Cadastro de funcionários, visitantes e veículos
- Entrada e saída de veículos
- Histórico de acessos
- CRUD completo integrado ao banco de dados

## Tecnologias
- Java
- Swing
- JDBC
- MySQL
- NetBeans

## Observações
Projeto acadêmico desenvolvido durante o curso técnico de informática.
